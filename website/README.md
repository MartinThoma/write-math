## Installation

### Composer

This website contains some third party packages. They are administrated via
composer.

Install composer:

```bash
curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer
```

Now run

```bash
composer update
```

### Misc

* Create `cache-data` folder.
